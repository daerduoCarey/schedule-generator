INSALLING JUnit
see http://junit.sourceforge.net/doc/faq/faq.htm#started_2

ADDING JUnit to a project in Eclipse
	select your project
	press the right mouse button
	select Properties
	select Java Build Path
	select Libraries
	press Add External JARs
	browse to your JUnit jar (junit-4.9b2.jar)
	press ok
	the jar should be added to your project
